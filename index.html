Sobre

Este portfólio reúne projetos de desenvolvimento em Python voltados para automação, integração com APIs públicas e soluções de apoio a processos educacionais. O objetivo é demonstrar experiência prática e consolidar ferramentas que podem ser aplicadas em cenários reais.

Projeto em Desenvolvimento – Automação com WhatsApp e IA

O projeto atual consiste na criação de um atendente virtual para salas de estudos, integrado ao WhatsApp e com uso de inteligência artificial.

  • Atendimento inicial a estudantes via WhatsApp;
  • Respostas automáticas a dúvidas frequentes;
  • Direcionamento a materiais de apoio e recursos;
  • Possibilidade de expansão para lembretes e avisos automáticos.

Portfólio de Projetos

  • Download CEIS – Portal da Transparência
    Script em Python que automatiza o download da base de dados do Cadastro Nacional de Empresas Inidôneas e Suspensas.
  • Extração de Dados do RGF – API Siconfi
    Script em Python que consulta a API oficial do Siconfi e consolida dados do Relatório de Gestão Fiscal em Excel.
